## View Small Text Files
```bash
cat <filename>
```

## View Files Using `less`
```bash
less <filename>
```
### Go Down     (`Space`)
### Go up       (`b`)
### Search      (`/<keyword>`)
### Next Match. (`n`)
### Exit        (`q`)

## Reset Terminal
```bash
reset
```

## Find File Type
```bash
file <filename>
```

## Open Image from Terminal
```bash
open <filename>
```
